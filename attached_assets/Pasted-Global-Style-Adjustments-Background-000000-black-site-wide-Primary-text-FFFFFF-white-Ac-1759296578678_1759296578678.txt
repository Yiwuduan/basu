Additional Details + Enhancements

Consistent spacing: use the 160px section padding as baseline—never drop below or overshoot without intention.

Line-length cap: body text lines should not exceed ~70 characters per line.

Scroll animations: each section should fade in and move upward ~20px on scroll, duration ~0.4s.

Button transitions: background / border / text should animate over 0.25s ease.

Input focus states: border glows orange (#FF4D00) on focus.

Image accenting: always overlay or accent images with a subtle orange element so they integrate with the palette (e.g. corner stripe, gradient fade, or highlight accent).

Typography variance: ensure headlines, subheads, and body text are visually distinct in size and weight—no two adjacent levels should feel too close.

Contrast checks: ensure text readability against background images (use overlay behind text if needed).
